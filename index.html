Tentang

Saya menggunakan nama samaran yaitu Grim, jadi saya boleh dipanggil Grim. Mengapa tidak menggunakan nama asli? Karena saya tidak mau nama saya terserbar saja walaupun cuma nama bukan privasi apa2 tetapi saya hanya tidak ingin saya di kenal banyak orang.

Pada website ini saya buat untuk menerangkan profil saya. Saya berasal dari Prov.Kepulauan Riau, Kab.Karimun, Kec.Kundur. Saya bersekolah di sini dan tamat juga disini dengan tamatan SMK Budhi Mulia Kundur. Untuk sekarang umur saya baru saja memasuki 17 Tahun, tetapi saya sudah kuliah maka beban yang akan diterima mungkin lebih banyak dari orang lain.

Saya juga sedang mencari pekerjaan yang tetap untuk tidak membebankan kedua orang tua saya. Ingin mencari kerja tetapi umur baru saja memasuki 17 dan masih dalam proses pembuatan KTP jadi masih harus tunngu. Karena kerja di perusahaan itu harus memiliki KTP akan tetapi saya belum punya. Jadi sekian dari saya saya ucapkan terima kasih jika ingin lebih jelas bisa memasukkan pesan di bawah atau melalui OnIg saya yang terletak pada bagian paling bawah.
